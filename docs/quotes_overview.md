Leaf's Quotes API returns crop futures prices from the Chicago Exchange (CBOT).

:::info
You can request in different currencies
:::

### Available conversions

We currently have the following conversions available:

| Crop       | Date                     |
|------------|--------------------------|
| corn       | 12/2020 |
| corn       | 03/2021 |
| corn       | 05/2021 |
| soybeans   | 12/2020 |
| soybeans   | 03/2021 |
| soybeans   | 05/2021 |
